# YouTube Ads Campaign Plan

## Todo List for Creating a YouTube Ads Campaign

[ ] Define campaign goals: Clearly outline what you want to achieve (e.g., increase brand awareness by 20%, generate 100 leads per week, drive $5000 in sales). Common goals include brand awareness (reach, views), consideration (views, engagement), action (conversions, sales), and subscriber growth.
[ ] Identify target audience: Determine demographics (age 25-45, gender, location), interests (fitness, technology, parenting), behaviors, and life events. Use Google Ads audience insights and YouTube Analytics to refine. Example: Target parents aged 30-45 interested in educational content for children.
[ ] Set up Google Ads account: Go to ads.google.com, sign in with Google account, follow setup wizard. Enter business information, billing details (credit card or bank account), and time zone/currency preferences. No minimum spend to start.
[ ] Link YouTube channel to Google Ads account: In Google Ads, go to Tools & Settings > Linked accounts > YouTube. Click "+" to link channel, sign in to the Google account that owns the YouTube channel, and approve the link. This enables video ad campaigns and access to YouTube analytics.
[ ] Create a new video campaign in Google Ads: Click "+ New Campaign", select goal (Sales, Leads, Website traffic, Brand awareness, Product consideration), choose "Video" as campaign type, and continue.
[ ] Choose campaign subtype: Select from Video reach (efficient reach), Video views (views or engagement), Product and brand consideration (consideration), or Drive conversions (website actions). For beginners, Video views or Product consideration are good starting points.
[ ] Set budget: Determine daily budget (e.g., $10-$50/day for testing) and total budget if applicable. Start small to test performance. Average CPV (cost-per-view) on YouTube ranges from $0.03 to $0.30 depending on targeting and competition.
[ ] Choose bidding strategy: Options include Maximum CPV (pay per view), Target CPM (pay per 1000 impressions), Target CPA (pay per conversion), or Maximize conversions. For view-focused campaigns, use Maximum CPV. For conversion-focused, use Target CPA or Maximize conversions.
[ ] Select ad formats: Choose which formats to use based on goals and budget. Skippable in-stream (TrueView): 12+ seconds, skippable after 5 seconds, pay when watched 30+ seconds or interacted. Non-skippable in-stream: 15-20 seconds, pay per impression. Bumper: 6 seconds non-skippable, pay per impression. Discovery: appears in search/related, pay per click.
[ ] For each ad format, note specifications: Video file (MP4, MOV, etc.), resolution (minimum 720p, recommended 1080p), aspect ratio (16:9, 4:3, 1:1, 9:16), maximum file size (1GB), length limits (varies by format), audio codec (AAC-LC recommended).
[ ] Create ad groups: Organize ads by theme, targeting, or format within the campaign. Each ad group contains one or more ads and shared targeting settings.
[ ] Set up demographic targeting: Select age ranges (18-24, 25-34, 35-44, 45-54, 55-64, 65+), gender, parental status, and household income (top 10%, 11-20%, 21-60%, lower 40%). Example: Target females 25-34 with high income for beauty products.
[ ] Set up topic targeting: Choose from thousands of topics (e.g., "Personal Finance", "Home Gardening", "Video Gaming") to show ads on videos about those subjects. Helps reach contextually relevant audiences.
[ ] Set up placement targeting: Select specific YouTube channels, videos, websites (Google Display Network), or apps where you want ads to appear. Example: Place ads on popular educational channels or specific competitor videos.
[ ] Set up keyword targeting: Choose words or phrases related to your product/service. Ads may show when users search for those terms on YouTube or watch videos containing those terms. Use Google Keyword Planner for research.
[ ] Set up affinity and in-market audiences: Affinity (long-term interests like "Sports Fans", "Travel Buffs"), In-market (actively researching/purchasing like "People shopping for laptops", "Car buyers"). Combines interest and purchase intent.
[ ] Upload video ad creative: Ensure video meets YouTube ad specs: Format (MP4 preferred), Codec (H.264), Audio (AAC-LC), Resolution (min 720x480, rec 1920x1080), Aspect ratio (16:9 ideal), File size (<1GB), Length (follow format rules). Enable captions for accessibility.
[ ] Add final URL, display URL, call-to-action, headline, description: Final URL = landing page URL. Display URL = shown in ad (can be simplified). CTA button options (Learn more, Shop now, Sign up, etc.). Headline (max 15 chars), Description (max 90 chars). Example: Headline "Learn Spanish", Description "Free trial today", CTA "Sign up".
[ ] Set up conversion tracking: Install Google tag (gtag.js) or Google Tag Manager on website. Create conversion action in Google Ads (Tools > Conversions). Import conversions from Google Analytics if needed. Track purchases, sign-ups, leads, etc.
[ ] Launch campaign: Review all settings, then click "Publish" or "Launch". Campaign typically starts delivering within hours after approval.
[ ] Monitor performance: Check metrics daily at first, then weekly. Key metrics: Views, View rate (views/impressions), Average CPV, Clicks, Click-through rate (CTR), Conversions, Cost per conversion. Use Google Ads dashboard and YouTube Analytics.
[ ] Optimize: Adjust bids based on performance (increase for good placements, decrease for poor). Refine targeting (exclude underperforming demographics/topics). Test different ad creatives (thumbnails, hooks, CTAs).
[ ] A/B test: Create experiments in Google Ads to test variables like bidding strategies, targeting options, or ad creatives. Run tests for sufficient duration (typically 2 weeks) with clear success metrics.
[ ] Report and iterate: Analyze results against goals. Scale successful campaigns by increasing budget or expanding targeting. Pause underperforming elements. Document learnings for future campaigns. Iterate every 2-4 weeks.
